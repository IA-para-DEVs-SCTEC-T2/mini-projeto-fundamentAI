Regulação CVM

A CVM regula quem pode emitir recomendações de investimento no Brasil. O produto precisa calibrar linguagem e escopo para operar como ferramenta educativa e informativa — não como consultoria de valores mobiliários sem habilitação. Disclaimers claros e linguagem neutra são obrigatórios.

Legal · Compliance
🪙
Gestão do consumo de tokens

Cada análise envolve múltiplas chamadas com contexto extenso — dados financeiros, macro e prompt estruturado. Sem roteamento por complexidade, prompt caching e limites por usuário, o custo escala rapidamente. A estratégia de modelo leve para consultas simples e modelo completo para análises profundas é essencial.

Custo · LLM
⚡
Latência e cache inteligente

5 a 8 chamadas de API sequenciais resultam em 10–15s de espera. A solução combina paralelização das chamadas de coleta, streaming da resposta do LLM e cache com TTL por tipo de dado — preços em minutos, fundamentos em dias, macro em semanas. Ativos frequentes como PETR4 e VALE3 devem ser pré-processados.

Performance · UX
🛡️
Alucinações e qualidade dos dados

O LLM pode gerar valores financeiros plausíveis mas incorretos se os dados não chegarem via ferramenta. Regra inviolável: todo número exibido vem das APIs — o modelo interpreta, nunca calcula. Validação cruzada entre fontes e data de referência visível em cada indicador são proteções fundamentais.
